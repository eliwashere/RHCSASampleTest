# RHCSASampleTest
Sample Test instructions and scoring scripts
The word document is a list of tasks to complete.
The test.sh runs all the other scripts.
Each other script are to test a given taks completion.

The scripts are written to allow for arguments to be passed so they can be used for different scenarios.


More to be planned.....

For now-
Download/clone the files 
put the scripts on the lab machine you are using
complete the tasks
run the test.sh to see which tasks you completed correctly.
